Title: A STAND-ALONE APPLICATION OF IMPLEMENTING LINEAR DATA STRUCTURES

Description: This project “A stand-alone application of implementing linear data structures” is all about performing all kinds of operations performed by the linear data structure.In the fig-1 is what the home page of the project looks like We can choose a data structure to perform the task by clicking on the different buttons for different operations. For example, when we click on the Array data structure it will be redirected to the page of the Array data structure. And there we can perform the task that we need. So, there are six different linear data structures used here. They are: 1. Array data structure 2. Stack data structure 3. Queue data structure 4. Circular data structure 5. Singly LinkedList 6.
